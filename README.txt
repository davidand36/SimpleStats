Simple Stats
Copyright (C) 2011 David M. Anderson

Simple Stats is an Android app designed to make common statistical tests easy. It is available on Google Play (formerly the Android Market): https://play.google.com/store/apps/details?id=us.EpsilonDelta.SimpleStats.

This app was built using Eclipse and the standard Android toolkit. All source code, build data, and libraries are in this repository. The source code for the javaEpsilonDelta_math library is archived separately at https://github.com/davidand36/javaEpsilonDelta.
